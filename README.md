# AI Skills for Reservoir Engineering
This project presents AI [skill](https://agentskills.io/home)s (for tools like Claude Code, Google Antigravity, etc) 
for (petroleum) reservoir engineering. It provides skills for the algorithms defined in the 
[pyrestoolbox](https://github.com/mwburgoyne/pyResToolbox) project.

The project is inspired by the [Claude for Reservoir Engineering](https://github.com/gabrielserrao/Claude-for-reservoir-engineering)
project, and provides the same facilities in a more flexible package: using [skill](https://agentskills.io/home)s instead of 
[MCP](https://modelcontextprotocol.io/), and not being limited to Claude as the AI tool.

## Approach
In addition to providing reservoir engineering skills for AI tools (a hopefully useful endeavor in its own right), 
this project also demonstrates an approch to convert [MCP](https://modelcontextprotocol.io/) servers into 
[Skill](https://agentskills.io/home)s.

## References
This work is based on the [pyResToolbox MCP Server](https://github.com/sanjaydasgupta/pyrestoolbox-mcp) project, 
and indirectly on the algorithms in the [pyrestoolbox](https://github.com/mwburgoyne/pyResToolbox) project.
